## Variable Names and Descriptions:

-   **variable**:

combined_data = the cleaned data to only include 50 states as well as the population data
er_data = the raw data set that is read in from GitHub
longest_visits = the 5 states with the longest ER visit times for mental health patients
n = the 5 states with the longest and shortest ER visit times for mental health patients
non_mental_health_patients = not used
p_data = the 5 states with the longest and shortest ER visit times for mental health patients including the full state name
P_data = the 5 states with the longest and shortest ER visit times for mental health patients
patients = Filtered data to only include mental health patient data
q1_data = not used
shortest_visits = the 5 states with the shortest ER visit times for mental health patients
state_populations = the states and their populations

## Data Types:

-   **Column**: 

state = the state abbreviation
condition = the care type, I only looked at Emergemcy Department
measure_id = ID for the different measure types
measure_name = The specific measure, for example average time patients spend in the ER
score = the qualitative value such as number of minutes a patient spent in the ER
footnote = not used
start_date = not used
end_date = not used
